# 📊 Análise de Dados Global: Caso Samsung (Didático)

Este projeto apresenta uma **Pipeline de Ciência de Dados** automatizada, desenvolvida para transformar dados brutos de vendas em um relatório executivo interativo. O foco principal é demonstrar competência técnica na extração de insights e visualização de dados voltada para Business Intelligence (BI).

## 🛠️ Tecnologias e Ferramentas
* **Python**: Linguagem base para o processamento.
* **Pandas**: Manipulação, limpeza e tratamento estatístico dos dados.
* **Plotly**: Criação de gráficos dinâmicos e interativos.
* **Estatística**: Cálculo de Coeficiente de Correlação de Pearson e Regressão Linear (OLS).
* **HTML/CSS**: Estruturação do layout do relatório final.

## 📈 O que foi analisado?
O script processa o dataset para responder a perguntas estratégicas de negócio:
1.  **Market Share Regional**: Distribuição percentual do faturamento por regiões globais.
2.  **Performance de Portfólio**: Ranking de categorias de produtos por receita gerada.
3.  **Elasticidade de Preço**: Análise da densidade de vendas em relação ao preço unitário (Heatmap).
4.  **Satisfação do Cliente**: Estudo da correlação entre o percentual de desconto aplicado e a nota de avaliação (Rating) do consumidor.

## 📝 Nota de Confidencialidade
Este repositório possui fins estritamente **didáticos**. Os dados e a marca citada são utilizados de forma fictícia para demonstração de portfólio, não representando informações reais ou internas da empresa.
